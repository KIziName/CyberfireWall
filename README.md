# Cyber_firewall-Pro
Assistant for Windows Firewall. Convenient configuration of IP and ports. Changing the black and white list. Monitoring active traffic packets to block unknown ones. All blocked IPs will also be blocked in Windows Firewall. The project is still in development, so there may be bugs and errors.


Помощник для брандмауэра Windows. Удобная настройка IP и портов. Изменение черного и белого списка. Мониторинг активных пакетов трафика для блокировки неизвестных. Все заблокированные IP-адреса также будут заблокированы в брандмауэре Windows. Проект все еще находится в разработке, поэтому возможны сбои и ошибки в работе.
